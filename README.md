# QyroBot | Discordメンバーバックアップ・自動参加認証Bot

Discordの**メンバーバックアップ（メン爆）**、**関連サーバーへの自動参加**、**認証ボット（Verify Bot）**機能を一つにまとめたツールです。

## 主な機能
- **メン爆（メンバーバックアップ）**: 認証済みユーザーを別サーバーへ一括追加
- **関連サーバー参加**: 認証完了と同時に予備サーバーへ自動参加
- **セキュリティ**: VPN/Proxyブロックによる安全な認証
